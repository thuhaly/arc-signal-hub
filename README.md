# Arc Signal Hub

A small Arc blockchain project for the Circle/USDC builder portfolio.

## Arc Network

- Chain ID: `5042002`
- RPC: `https://rpc.testnet.arc.network`
- Explorer: `https://testnet.arcscan.app`
- Gas token: `USDC` with 6 decimals

## Features

- Pure JavaScript module for Arc/USDC workflows
- Built-in Node.js test suite
- No private keys or secrets committed

## Commands

```bash
npm test
npm run build
```
